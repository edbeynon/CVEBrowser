# CVEBrowser
This project is for tooling to aid in yocto development.

The initial release is CVEBrowser.

All processing is in the browser, nothing is uploaded anywhere.

Tested in chrome.

This is strictly an early Beta release  and is not tested for all possible yocto layers by any stretch of the imagination!

Just load it into your browser and point it at the directory with your CVE files. It reads json.

Feedback appreciated!
